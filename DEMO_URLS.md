# Solar Scout — White-Label Demo URLs

Paste any of these into your browser to preview white-label configurations.

## Greek hotel (English)
```
/?hotel=Villa+Aegean&color=%23E8834B&lang=en&logo=https://ui-avatars.com/api/?name=Villa+Aegean&background=E8834B&color=fff&size=128&rounded=true
```

## German hotel (German)
```
/?hotel=Strandhotel+R%C3%BCgen&color=%232E86AB&lang=de&logo=https://ui-avatars.com/api/?name=Strandhotel+R%C3%BCgen&background=2E86AB&color=fff&size=128&rounded=true
```

## Spanish hotel (English)
```
/?hotel=Hotel+Sol+Marbella&color=%23C0392B&lang=en&logo=https://ui-avatars.com/api/?name=Hotel+Sol+Marbella&background=C0392B&color=fff&size=128&rounded=true
```

---

## Parameters reference

| Param    | Description                              | Example                  |
|----------|------------------------------------------|--------------------------|
| `hotel`  | Hotel display name                       | `Villa+Aegean`           |
| `logo`   | Full URL to hotel logo image             | `https://...`            |
| `color`  | Brand hex color (encode `#` as `%23`)    | `%23E8834B`              |
| `lang`   | Language: `en`, `de`, or `gr`           | `de`                     |

Omit all params to show default Solar Scout branding.
